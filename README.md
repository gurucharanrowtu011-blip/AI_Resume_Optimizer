



 📄 AI Resume Optimizer (GenAI + Prompt Engineering)



 🚀 Project Overview

This project is a **Streamlit-based AI Resume Optimizer** that enhances raw resumes into professional, structured, and ATS-friendly formats using **Prompt Engineering techniques and rule-based NLP simulation (offline GenAI system)**.

Unlike traditional ML projects, this system focuses on **prompt-driven resume transformation and intelligent text restructuring without requiring external APIs or internet connectivity**.



 🎯 Features

* 📄 Resume text input interface
* 🔄 Three optimization modes:

  * Basic Improvement
  * ATS Optimization
  * Role-Based Optimization
* 📊 Before vs After Resume comparison
* 🎯 ATS Score estimation system
* ⚡ Fully offline (no API or internet required)
* 🧠 Prompt Engineering-based logic simulation
* 💻 Built using Streamlit for interactive UI



 🛠️ Tech Stack

* Python 🐍
* Streamlit 🎈
* NLP (Rule-based text processing)
* Prompt Engineering Concepts

 ⚙️ How It Works

 1️⃣ Input Resume

User enters raw resume text in the Streamlit UI.

 2️⃣ Select Mode

User selects one of:

* Basic Improvement
* ATS Optimization
* Role-Based Optimization

 3️⃣ Processing

The system applies:

* Text cleaning
* Keyword enhancement
* Structured formatting
* Role-based modifications

 4️⃣ Output

* Improved resume
* ATS score
* Before vs After comparison



 📊 ATS Score Logic

A simple keyword-based scoring system:

* Python
* Machine Learning
* SQL
* Data Analysis
* Java / C++

Each keyword contributes to the final ATS score (0–100).



 🧠 Key Concept

This project demonstrates:

> **Prompt Engineering + Rule-Based AI Simulation**

Instead of using external LLM APIs, it mimics AI behavior using structured transformation rules and intelligent prompt design logic.
